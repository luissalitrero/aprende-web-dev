Curso de HTML5

Bienvenido al curso de HTML5, aquí aprenderás lo necesario para construir tus aplicaciones web, que es el lenguaje HTML, actualmente en su versión 5.

Nota: En este curso utilizaremos el término "aplicación web" en lugar de "página web", que en escencia son lo mismo, ya que podríamos decir que una aplicación web es una página web pero con funcionalidad.

HTML significa HyperText Markup Language, que se traduciría como Lenguaje de marcado de hypertexto, el cual es el lenguaje utilizado para crear nuestras aplicaciones web. Dicho lenguaje consiste en un conjunto de etiquetas, las cuales nos servirán para definir los elementos y la estructura de la aplicación web.

Una etiqueta es un conjunto de letras y números, rodeados por los símbolos de "menor que" < y "mayor que" >. La mayoría de las etiquetas consisten en una etiqueta que abre y otra que cierra, para poner en medio de ellas sus contenidos, aunque hay etiquetas que no necesitan una etiqueta de cierre ya que no contienen nada. Ejemplos:

Con etiqueta de cierre:

La etiqueta <b></b> hará que el texto dentro de las etiquetas de apertura y de cierre sea mostrado en negritas.

<b>contenido de la etiqueta</b>.
Sin etiqueta de cierre:

La etiqueta <br /> agrega un salto de línea donde es colocada, así el texto que le sigue se mostrará en una nueva línea.

<br />
